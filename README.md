# The Beach

Based on the [Conditional Design Workshop](https://conditionaldesign.org/workshops/the-beach/)

This is a Human//Computer turn-by-turn project.

A user draws a dot on a piece of paper, and the AxiDraw responds with a dot of its own.
By using a webcam to track the dot, the system can create a feedback loop between the user and the machine.

## Python AxiDraw API install

```bash
python -m pip install <https://cdn.evilmadscientist.com/dl/ad/public/AxiDraw_API.zip>
```
